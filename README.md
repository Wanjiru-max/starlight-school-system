# 📘 Starlight Institute School Management System

A full-stack web-based School Management System designed to streamline academic and administrative operations at **Starlight Institute** through automation, role-based access, and efficient data management.

---

## 🚀 Overview

This system provides a centralized platform for managing students, teachers, courses, enrollments, and academic results. It reduces manual paperwork, improves accuracy, and enhances efficiency in school administration.

---

## ⭐ Key Features

- 🔐 Role-based authentication (Admin & Teacher)
- 👨‍🎓 Student management (Add, Edit, Delete)
- 👩‍🏫 Teacher management system
- 📚 Course and program management
- 📝 Student enrollment system with validation
- 🚫 Prevention of duplicate enrollments
- 📊 Exam results entry and management
- 📄 Enrollment reports with filtering options
- ✏️ Edit and delete functionality with confirmation prompts

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS  
- Backend: PHP  
- Database: MySQL  
- Server Environment: XAMPP / WAMP
